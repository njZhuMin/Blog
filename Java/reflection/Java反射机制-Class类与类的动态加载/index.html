我们当然可以像最开始那样使用if条件判断，但是这样带来的问题就是每次我们添加一个功能都需要重新更改OfficeBetter的代码和重新编译。其实我们只需要定义一个接口类， 让各个子模块实现接口类即可实现动态加载，而且不需要改变Office类的实现。
